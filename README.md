# DocxToWhatsapp
A small python script that allows me to open a .docx file and do some modifications that will allow me to copy &amp; paste in Whatsapp

# Versions
Actual version : 3
- 1- Basic
- 2- GUI + copy&paste
- 3- Allow an exe file

# Dependancy
- python : pyperclip
